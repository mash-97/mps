# MPS (MonoPsyches)

MPS Manager in short!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mps'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install mps

## Usage

Create user.
Create MPS.
Read MPS.
Delete MPS.
Edit MPS.
And F**k MPS..
